An AI-powered personal wellness coach delivered through Telegram — proactive daily check-ins, 24/7 on-demand support, and automated exercise tracking, all in a single n8n workflow.

Overview

Most wellness apps fail because they require behavior change before delivering value — users must remember to open them. Wellness Coach Buddy flips this by meeting users where they already are: Telegram. No app to install, no new habits to form.

Built as a single n8n workflow with dual trigger paths, the bot proactively sends morning and evening check-ins on a schedule, responds to on-demand messages 24/7, and automatically logs exercise habits through keyword detection — all powered by three specialized Claude agents with 10-turn conversation memory.

How It Works
1
Dual Trigger Paths Schedule triggers fire at 7:00 AM (morning check-in) and 9:00 PM (evening reflection). A separate webhook trigger activates when the user sends a message.
2
Central Switch Router Both paths converge at a Switch node that routes traffic to the appropriate agent — morning, evening, or on-demand — based on trigger source and time.
3
AI Agent Processing Each Claude Sonnet agent maintains a 10-turn conversation window. The on-demand agent also scans messages for exercise keywords to log workout activity.
4
Exercise Tracking & Deduplication Detected exercise logs are written to n8n Data Tables with deduplication logic to prevent double-counting the same workout within a session.
5
Weekly Progress Nudges The system tracks against a 4x/week exercise goal and surfaces progress during evening check-ins to maintain accountability.

Architecture Notes
Single workflow design. The entire system — scheduling, routing, AI processing, data persistence, and delivery — runs as one n8n workflow. Two independent trigger paths converge at a central Switch router, keeping the architecture simple and the execution path predictable.
~$2/month to run. n8n self-hosted is free. Claude API usage for personal coaching conversations costs approximately $1–2/month, making this a near-zero-cost production system.
