LaRis: Headline Analyzer for Better SEO Visibility

Website, Github

LaRis merupakan sistem untuk menganalisis judul yang bertujuan untuk dapat mengoptimalkan SEO. Pada sistem LaRis judul yang telah ditambahkan akan dilakukan analisis sentiment, analisis emotional dan juga komparasi dengan hasil pencarian langsung berdasarkan google search engine. Beberapa library yang digunakan dalam pengembangan laris adalah Scikit-learn, Tensorflow, Beautifullsoap4, dan Requests. Untuk mengetahui informasi lebih detail dapat dilihat pada link github dan website diatas.

Skincare Review Rating Classifier

Dashboard, Github

Skincare Rating Prediction merupakan project NLP yang bertujuan untuk memprediksi rating yang diberikan oleh user berdasarkan review produk yang diberikan. Dataset yang digunakan pada project ini merupakan hasil scrapping review salah satu produk sunscreen lokal dari website femaledaily.com, librarY yang digunakan dalam project ini adalah Seaborn, Matplotlib, Pandas, Scikit-learn dan Tensorflow.

HR Attrition Analysis and Prediction

Dashboard, Github

Objective pada tugas ini adalah menklasifikasikan apakah seorang karyawan dengan karakteristik tertentu berdasarkan data apakah cenderung Attrition atau tidak. Attrition sendiri adalah kondisi dimana karyawan akan meninggalkan perusahaan dengan berbagai alasan seperti resign, pensiun dan lain sebagainya. Pada tugas ini dataset yang digunakan berasal dari Kaggle dengan judul IBM HR Analytics Employee Attrition & Performance

Analysis Spread of Covid-19 in Indonesia

Dashboard

Project ini merupakah dashboard Covid-19 dengan menggunakan data perkembangan covid-19 di Indonesia sejak awal maret hingga 23 Agustus 2021. Dashboard ini dibuat dengan tujuan melihat perkembangan covid-19 di Indonesia, vaksinasi dan perkembangan berdasarkan provinsi. Package yang digunakan dalam pembuatan dashboard ini adalah Streamlit, Plotly dan Pandas untuk data processing. Untuk kode lengkapnya dapat di lihat pada daftar link diatas.
